# 🚀 C++ Fundamentals & Logic Repository

Welcome to my personal C++ development repository! This repository serves as a comprehensive lab manual and code showcase, documenting my journey through core programming logic, array manipulation, and low-level computer science concepts (including custom binary/decimal mathematical converters).

All programs are written in clean, standard C++ and are designed to demonstrate foundational algorithmic thinking.

---

## 🛠️ Program Directory

| Program Name | Key Concept / Tech Stack | Functional Description |
| :--- | :--- | :--- |
| 📁 `arrayintro.cpp` | **1D Arrays** | A foundational introductory program demonstrating basic array declaration, initialization, and element traversal. |
| 📁 `arraysearch.cpp` | **Search Algorithms** | Simulates a database check by searching a static array to verify if a user-inputted record exists. |
| 📁 `bin2dec.cpp` | **Number Systems & Math** | Independently engineered mathematical decoder that parses raw binary inputs and converts them to base-10 Decimal format. |
| 📁 `chess.cpp` | **Nested Loops & Matrices** | Generates a 2D grid matrix printing alternating `0` and `1` values to simulate a chess board layout. |
| 📁 `converter.cpp` | **Custom Functions & Bitwise Prep** | Independently developed application that converts base-10 Decimal integers into raw Binary blocks using automated custom modular functions. |
| 📁 `dec2bin.cpp` | **Iterative Division Logic** | An alternative Decimal-to-Binary conversion engine utilizing an array stacking mechanism, developed via collaborative AI pairing. |
| 📁 `oddoreven.cpp` | **Conditional Control Flow** | Iterates from numbers 1 through 50, executing conditional runtime logic to explicitly tag and output each integer as either Odd or Even. |
| 📁 `primechecker.cpp` | **Algorithmic Primality** | Evaluates a standalone user-provided integer to mathematically verify whether it is a Prime number. |
| 📁 `primefinder.cpp` | **Looping Prime Evaluation** | Scans an entire numeric range from 1 to 50, computing and displaying the primality status of each individual value. |
| 📁 `rating.cpp` | **Data Accumulation & Arrays** | Captures multiple dynamic user ratings, stores them sequentially within an array structure, and calculates the precise mathematical average. |
| 📁 `squareprinter.cpp` | **Pattern Generation** | Utilizes multi-layered nested loops to render a perfectly formatted $5 \times 5$ square matrix of characters (`*`). |
| 📁 `table.cpp` | **Dynamic Loops** | Generates and prints a complete, formatted mathematical multiplication table for any specific number provided by the user. |
| 📁 `table2.cpp` | **Matrix Printing** | Automatically maps and builds sequential mathematical tables for all integers starting from 1 up to 10. |
| 📁 `weather.cpp` | **Array Analysis & Stats** | Tracks a full 7-day temperature log within an array, computing the highest, lowest, and absolute average recorded Celsius values. |

> ⚠️ *Note: The `test.cpp` file present in the repository is used strictly for sandbox experimentation and environment testing; it can be safely disregarded.*
